About drift

Drift is a multi-agent stress-test simulator. It puts a small team of AI agents into a fake environment that changes over time, and watches for failures that only show up when agents work together — contradictions, loops, hallucinated references, coordination failures across timesteps.

This UI runs locally against the same Python codebase you'd use from the CLI. Runs are written to runs/ as JSONL and can be replayed or compared.
