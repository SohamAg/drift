Bring your own agents — rewiring

The @drift.agent decorator + Custom tab predated the LangGraph adapter. After the 2026-06-29 cleanup the native simulator was removed, so this tab is paused while we re-wire it to run user-supplied agents through the adapter path.

The decorator @drift.agent still imports as a data shape; only the runtime path is paused. Want to test your agents now? Wire them into a LangGraph and use the Adapter tab, or call drift_test from a Python script (see examples/adapters/).
