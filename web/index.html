Bring your own agents + environment

Define one or more @drift.agent async functions in the editor below. Optionally subclass drift.WorldState for your domain, define an initial_state() function to populate it, and/or an events() function returning [(timestep, event), …] for scheduled chaos. See the loaded example for the full shape.

Executes Python in the local server process. Safe for local development; do not expose this server to untrusted users.

Drift walks your WorldState fields and injects type-appropriate mutations (bool flips, dict clears, list reverses, etc.) at random steps. Injected events appear in results prefixed AutoChaos.<pattern>[field].

Adds an LLM-judged detector alongside the deterministic ones. Findings are prefixed llm: across five families: coordination_contradiction, grounding_failure, state_drift, emergent_decay, gate_bypass.

Plain-English patterns the judge will additionally watch for. Matches are reported under llm:user_guideline:<n>. Use for domain-specific patterns the built-in five-family taxonomy doesn't cover.
